# LearnMate AI 

LearnMate AI is a local Retrieval-Augmented Generation (RAG) system designed to help students interact intelligently with PDF documents.


## LearnMate AI Modules:

1. Document Processing Module
2. Embedding Module
3. Vector Database Module
4. RAG Question Answering Module
5. Summarization Module
6. Quiz Generation Module
7. Mind Map Module
8. Audio Generation Module
9. User Interface Module
10. Multi-Document Manager

and others...
